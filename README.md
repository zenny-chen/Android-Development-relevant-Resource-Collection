macOS下adb工具所在路径：`~/Library/Android/sdk/platform-tools/adb`

如果用NDK编译C艹代码，那么需要在`Application.mk`中添加`APP_STL := gnustl_static`，否则大部分C艹标准库都无法找到。而如果要用当前LLVM的runtime的STL库的话，可以使用：`APP_STL := c++_static`。
